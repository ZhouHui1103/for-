## 业务流程指标

| 业务流程 | 类型 | 过程指标 (状态演变及周期时长) | 流程结果指标 | 流程财务指标 |
| :--- | :--- | :--- | :--- | :--- |
| 1.0 订单承接与分配 | 业务流程 | **状态演变**: `待分配 -> 交付中 -> 服务准备就绪` <br> **周期时长**: 订单分配平均耗时 | 服务进程创建成功率 | - |
| 2.0 服务启动与Onboarding | 业务流程 | **状态演变**: `待开启 -> 联系建立 -> 进程中` <br> **周期时长**: Onboarding平均完成耗时 | 首次反馈获取率 | - |
| 3.0 日常督导循环 | 业务流程 | **状态演变**: `待完成 -> 已完成` (每日循环) <br> **周期时长**: 每日待办平均完成耗时 | 每日待办完成率、每日进度记录率 | - |
| 4.0 客户请假处理 | 业务流程 | **状态演变**: `申请中 -> 已批准` <br> **周期时长**: 请假平均审批耗时 | 服务时间线调整准确率 | - |
| 5.0 定期质量评估 | 业务流程 | **状态演变**: `评估周期到达 -> 数据收集 -> 已评估` <br> **周期时长**: 平均评估耗时 | 评估覆盖率 | - |
| 6.0 客户反馈收集 | 业务流程 | **状态演变**: `关键节点到达 -> 待操作 -> 已操作` <br> **周期时长**: 反馈收集平均耗时 | 关键节点反馈获取率 | - |
| 7.0 服务续费与结束 | 业务流程 | **状态演变**: `准备结束 -> 已结束` <br> **周期时长**: 结束流程平均耗时 | 续费意向沟通率、客户续费转化率 | 续费订单金额 |

## 角色薪酬绩效财务指标

| 角色       | 财务指标                                                                                                                                                         |
| :--------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `支撑运营` | 绩效奖金与服务进程的按时启动率、客户满意度、流程执行准确率挂钩 (间接影响运营成本和效率)                                                                        |
| `督导师`   | 基础服务费、绩效奖金、复购激励与客户满意度、每日计划完成率、复购率挂钩 (直接与服务成本、客户留存和复购收入相关)                                                    |
| `培训师`   | 管理奖金与所负责小队的整体服务质量、督导师成长情况、客户复购率挂钩 (间接影响服务质量和客户复购收入)                                                              |
| `客户`     | 不适用 (客户为付费方)                                                                                                                                            |
