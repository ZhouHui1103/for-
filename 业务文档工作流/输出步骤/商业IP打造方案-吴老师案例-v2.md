# 商业IP打造方案-吴老师案例-v2

**L0-IP画像卡 (摘要)**
- **IP名称**: 吴老师
- **商业模式类型**: 实体招生
- **产品/链路复杂度**: 单点转化 (核心目标是报名入学)
- **特殊限制/背景**: 公立学校背景，不允许直接线上成交；每年仅有春秋两季招生窗口。

---

## 一、产品与成交路径表

| 项目 | 具体内容填写 (基于客观事实) |
| :--- | :--- |
| **产品/服务名称** | 工匠学院模式 (产教融合、赛教融合的实践教育体系) |
| **产品定义** | 一个为中考失利或寻求差异化成才路径的学生，提供以真实企业项目和技能竞赛为核心，旨在实现高质量就业的职业教育体系。 |
| **目标用户群体** | 15-18岁学生及其家长。 |
| **价格/价位带** | 公办学校学费标准。 |
| **核心卖点/被认可点** | ① **结果可量化**: 学生毕业后平均手握3个offer（“1:3选择权”），能力和薪资超越部分本科生。<br>② **培养模式独特**: 真正的“产教融合”，提供大量京东、阿里等级别的真实企业项目，让“所学即所用”。<br>③ **理念深入人心**: “托举式”教育，能“看见每个孩子”，发掘其独特亮点，实现“弯道超车”。 |
| **用户来源** | ① 线下常规招生；② 家长及毕业生口碑推荐；③ 人社局等官方渠道分配。 |
| **成交方式/链路** | **公域(短视频) -> 私域(社群/咨询) -> 线下(开放日/分享会) -> 报名入学** |
| **被吐槽点** | （录音中未明确提及，但可推断为社会对“职业教育”的普遍偏见） |

## 六、成交链路执行步骤图（文字版）

1.  **引流环节**: 用户通过 **抖音/视频号短视频**（例：“吴老师人生故事”、“学生逆袭案例”）看到内容 → 被 **“为被放弃的孩子找到出路”的情感共鸣** 和 **“超越本科生”的真实结果** 吸引；
2.  **承接环节**: 用户通过 **评论区引导/私信回复** 添加官方微信号/社群 → 添加后收到 **欢迎语+1-2个相关的学生成功案例视频**；
3.  **培育环节**: 私域内通过 **日常教学视频/校企合作动态/教育理念分享** 培育信任 → 持续 **每周1次主题直播/2-3次深度图文** 触达；
4.  **转化环节**: 通过 **线上/线下开放日/教育理念分享会** 引导深度了解 → 最终引导至招生老师进行 **“一对一报名咨询”**；
5.  **裂变环节**: 通过 **毕业生返校分享会/家长“推荐官”计划** 引导口碑裂变。

## 九、内容定制与落地准备表

| 准备项目 | 具体内容填写 |
| :--- | :--- |
| **IP定妆** | **形象风格**: 专业严谨的教育家形象与温和亲切的长者形象相结合。<br>**拍摄场景**: 办公室（背景有书籍、学生作品）、实训室（与学生互动）、教室（授课状态）。 |
| **内容排期 (MVP)** | **破冰期 (1-4条)**：1条人生故事（3-5分钟版）+ 3条核心理念金句短视频。<br>**信任期 (5-20条)**：5-8条不同类型的学生“逆袭”案例 + 5-8条教育痛点/焦虑话题解读。 |
| **流量类型匹配** | **泛流量内容**: “放弃百万年薪当老师”人生故事、#就业是最大的民生 等社会热点话题解读。<br>**精准流量内容**: “孩子沉迷游戏怎么办”、“成绩不好如何弯道超车”等具体问题解决方案、学生案例。 |
| **IP表达力分析** | **优势**: 真诚、有感染力、眼里有光、教育理念自成体系、金句频出。<br>**待提升**: 镜头感和标准化的表达节奏可能需要专业引导和练习。 |
| **变现准备** | **是否可直接变现**: **否，需强引导** / **补充**: 因IP身份限制，所有公域内容终点是引导添加私域，由招生老师团队承接转化。需为招生团队准备标准化的咨询流程SOP和素材包。 |
