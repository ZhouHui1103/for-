模块实验室（Module Labs）- Vibe Coder 职位描述

模块实验室（Module Labs）是风变科技孵化的新型AI组织。我们秉持“万物皆模块”的工程哲学，致力于推动传统服务业的AI化与工业化变革。我们深入企业业务一线，将复杂的人力密集型流程重塑为标准化、高效率的AI工程化解决方案，以此释放全新的商业价值。

【Vibe Coder：新时代的业务架构师】
Vibe Coder 是AI时代应运而生的新型角色，其核心并非传统的编程技能，而是卓越的业务理解力与逻辑构建能力。你将专注于AI Coding，深入理解商业问题的本质，通过流程抽象，并运用前沿AI工具（如Claude Code, n8n, 多维表格等）构建并交付自动化解决方案。在这里，你将是连接商业需求与AI技术的关键桥梁。

【核心职责】
- **业务流程洞察：** 深入客户业务场景，精准识别、梳理并分析核心业务流程与关键节点。
- **模块化抽象：** 运用系统性思维，将复杂的业务流程解构为标准、可复用的功能模块。
- **AI解决方案构建：** 采用Vibe Coding范式，主导业务的AI智能化与自动化改造，创造并交付可直接衡量商业价值的解决方案。

【任职要求】
- **系统性思考者：** 具备出色的抽象思维与逻辑拆解能力，能够洞察复杂系统背后的结构与规律。拥有工程、计算机或相关理科背景者优先。
- **积极的创造者：** 对利用AI技术改造和优化商业流程抱有浓厚兴趣，具备从0到1的项目经验和强烈的自驱力。
- **熟练的实践者：** 拥有Vibe Coding、低代码/自动化工作流（如n8n）的实践经验，或具备多维表格等工具的高级应用能力。

【你将获得】
- **前沿的方法论与实践：** 获得一线的AI工程化项目经验，在资深导师的1对1指导下，快速提升专业能力与认知高度。
- **“一人公司”的成长路径：** 从入职第一天起，你便是一位独立的商业价值创造者。我们将为你提供真实的商业项目，并提供全方位支持，助你构建能持续创造收入的个人事业。
- **灵活高效的工作模式：**
    - **工作方式：** 远程协作，实行灵活的任务/项目制。
    - **薪酬回报：** 提供行业内极具竞争力的项目分成机制（月收入最高可达2万元），让你的贡献获得应有回报。
    - **工具支持：** 提供Claude Code 20x Max等业界顶级的AI工具，最大化你的创造效率。
