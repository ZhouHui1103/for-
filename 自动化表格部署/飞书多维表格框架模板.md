# 飞书多维表格框架模板.md

**版本**: 1.0
**日期**: 2025-09-23
**说明**: 本模板旨在为新业务的快速数据建模提供一个通用起点。请根据实际业务需求，替换 `[占位符]` 内容并增删表格与字段。

---

## 数据库表结构设计原则

- **核心实体表**: 业务中的核心名词，如客户、产品、员工。数据相对长期稳定。
- **流程驱动表**: 记录业务动作的表，如订单、项目、任务。数据有明确的生命周期和状态变化。
- **过程记录表**: 在流程中产生的各类记录，如日志、反馈、审批记录。
- **资源配置表**: 用于管理和分配业务所需资源的表，如团队、设备。

### **核心实体表**

#### 1. [核心实体A] 信息表 (示例: 客户表)
| 字段名称 | 字段类型 | 选项 / 说明 |
| :--- | :--- | :--- |
| **[实体A ID]** | **文本 / 自动编号** | **(主字段)** 实体的唯一标识。 |
| **状态** | **单选** | `[状态A]`, `[状态B]`, `[状态C]` |
| **[分类维度X]** | **多选** | `[选项A]`, `[选项B]`, `[选项C]` |
| **[分类维度Y]** | **单选** | `[选项D]`, `[选项E]` |
| **关联[流程驱动表]** | **关联** | 关联到 `[核心流程]跟踪表`。 |
| **创建时间** | **创建时间** | |

#### 2. [核心实体B] 信息表 (示例: 员工表)
| 字段名称 | 字段类型 | 选项 / 说明 |
| :--- | :--- | :--- |
| **[实体B名称]** | **文本** | **(主字段)** |
| **角色/岗位** | **单选** | `[角色A]`, `[角色B]` |
| **状态** | **单选** | `[在职]`, `[离职]` |
| **关联[资源配置表]** | **关联** | 关联到 `[资源单元]配置表`。 |

### **流程驱动表**

#### 3. [核心流程] 跟踪表 (示例: 服务进程表)
| 字段名称 | 字段类型 | 选项 / 说明 |
| :--- | :--- | :--- |
| **[流程ID]** | **自动编号** | **(主字段)** |
| **关联[核心实体A]** | **关联** | 关联到 `[核心实体A]信息表`。 |
| **[实体A名称] (带出)** | **查找** | 从 `关联[核心实体A]` 中自动查找并带出名称。 |
| **关联[资源配置表]** | **关联** | 分配负责的 `[资源单元]`。 |
| **状态** | **单选** | `[待开始]`, `[进行中]`, `[已完成]` |
| **开始时间** | **日期** | |
| **计划结束时间** | **日期** | |
| **关联[过程记录表]** | **关联** | 关联到 `[原子任务]记录表` 等。 |

---

### **资源配置表**

#### 4. [资源单元] 配置表 (示例: 团队表)
| 字段名称 | 字段类型 | 选项 / 说明 |
| :--- | :--- | :--- |
| **[资源单元名称]** | **文本** | **(主字段)** |
| **负责人** | **关联** | 关联到 `[核心实体B]信息表`。 |
| **成员列表** | **关联** | 关联到 `[核心实体B]信息表` (允许多个)。 |
| **总容量** | **数字** | 该单元可承载的任务上限。 |
| **关联[核心流程]** | **关联** | 关联正在执行的 `[核心流程]跟踪表`。 |
| **已分配数量** | **汇总** | **公式**: `COUNT(关联[核心流程])` |
| **剩余可用** | **公式** | **公式**: `{总容量} - {已分配数量}` |
