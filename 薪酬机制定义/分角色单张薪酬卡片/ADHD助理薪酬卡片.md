# ADHD助理薪酬卡片

*   **版本**: V1.0
*   **更新日期**: 2025-10-13
*   **文档说明**: 本文档基于《薪酬卡片文档写作规范与模板》及《ADHD助理薪酬机制》文档，为ADHD助理角色提供清晰、可执行的薪酬方案。

---

### 一、 指标定义

*   **赞藏评总和**: 在一个自然周内，单个账号发布的所有内容的“点赞数”、“收藏数”、“评论数”的总和。这是评定账号等级的核心依据。数据来源于各内容平台后台或数据中台。
*   **账号等级**: 每周评定一次，基于账号当周的`赞藏评总和`。等级分为S、A、B、C四级。
*   **直接订单数量**: 在一个考核周期（月度）内，通过该助理运营的账号直接带来的、已确认成交的订单总数。数据来源于`订单信息表`。

---

### 二、 薪酬元规则 (Compensation Meta-rules)

#### **`XC-6-1`：助理业绩提成规则**
1.  **规则**：根据助理在一个月内完成的`直接订单数量`，计算其业绩提成。
2.  **概念定义**：
    *   `成交订单`：指在`订单信息表`中，已确认支付并归属于该助理的订单记录。
3.  **参数**：
    *   `参数1`：单笔订单提成金额（元/单）。
4.  **计算方法**：
    *   `薪酬 = 参数1 * 直接订单数量`
5.  **数据来源**：`订单信息表`
6.  **相关注释**：此规则直接激励最终的商业转化成果。

#### **`XC-6-2`：助理等级维护激励规则**
1.  **规则**：根据助理负责的账号每周评定的`账号等级`，发放对应的周度激励奖金。
2.  **概念定义**：
    *   `周度评级`：指在每周固定时间点，依据《ADHD助理薪酬机制》中的评级标准对账号进行的等级评定。
3.  **参数**：
    *   `参数1`：S级账号周度激励（元/周）。
    *   `参数2`：A级账号周度激励（元/周）。
    *   `参数3`：B级账号周度激励（元/周）。
    *   `参数4`：C级账号周度激励（元/周）。
4.  **计算方法**：
    *   `月度激励 = Σ (每周评定等级对应的激励奖金)`
5.  **数据来源**：`ADHD账号评级规则`、项目周报/数据中台
6.  **相关注释**：此规则激励助理持续产出高质量内容，保持账号活跃度和用户粘性。

### 三、 薪酬卡片 (Salary Cards)

#### **ADHD助理 薪酬卡片**
*   **项目**：ADHD督导业务
*   **角色**：助理（铺量团队）

**1. 薪酬构成**：
*   **业绩提成**: 引用元规则 `XC-6-1`；`参数1` = `50` 元/单。
*   **等级维护激励**: 引用元规则 `XC-6-2`，参数设定如下：
    *   `参数1` (S级) = `100` 元/周
    *   `参数2` (A级) = `60` 元/周
    *   `参数3` (B级) = `20` 元/周
    *   `参数4` (C级) = `0` 元/周

**2. 考核规则**：
*   **内容发布量**: 每周至少发布 5 条符合规范的内容。
*   **账号健康度**: 月度内账号等级不得连续两周为 C 级。
*   **数据回传及时性**: 所有要求的数据需在规定时间内完成回传，延迟次数每月不得超过 2 次。

### 四、 核心指标与薪酬测算

本章节旨在基于已定义的薪酬规则，对ADHD助理岗位的月度薪酬进行合理性测算。

#### **1. 单节点月度到手薪酬测算**

*   **ADHD助理到手薪酬** = 
    ( `Σ 每周等级维护激励` ) + 
    ( `月度直接订单数量` × 50 )
    
    *   **示例**: 一位ADHD助理在一个月（假设为4周）内，其工作业绩如下：
        *   **周度账号评级**:
            *   第一周: A级 (赞藏评总和 600)
            *   第二周: S级 (赞藏评总和 950)
            *   第三周: A级 (赞藏评总和 700)
            *   第四周: B级 (赞藏评总和 400)
        *   **月度业绩**: 产生了 `5` 笔直接订单。
    *   **薪酬计算**:
        *   **等级维护激励**: 60 (A级) + 100 (S级) + 60 (A级) + 20 (B级) = **240元**
        *   **业绩提成**: 5 × 50 = **250元**
        *   **合计月度薪酬**: 240 + 250 = **490元**
    *   **说明**：此薪酬为基础薪酬，不包含可能存在的其他项目奖金或特殊激励。

### 五、 (附录) 账号等级评定规则

账号等级根据**账号每周发布所有内容的“赞、藏、评”总和**进行评定。

*   **S级账号**:
    *   **标准**: 每周“赞藏评”总和 ≥ 800

*   **A级账号**:
    *   **标准**: 500 ≤ 每周“赞藏评”总和 < 800

*   **B级账号**:
    *   **标准**: 150 ≤ 每周“赞藏评”总和 < 500

*   **C级账号**:
    *   **标准**: 每周“赞藏评”总和 < 150
