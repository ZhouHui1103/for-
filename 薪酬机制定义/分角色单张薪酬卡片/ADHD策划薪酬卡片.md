# ADHD策划（培训节点）薪酬卡片

*   **版本**: V1.0
*   **更新日期**: 2025-10-13
*   **文档说明**: 本文档基于《薪酬卡片文档写作规范与模板》及《ADHD策划薪酬机制》文档，为ADHD策划（培训节点）角色提供清晰、可执行的薪酬方案。

---

### 一、 指标定义

*   **整体团订单数量**: 在一个考核周期（月度）内，该策划所带领的整个团队（所有助理）完成的`直接订单数量`的总和。数据来源于`订单信息表`。
*   **团队账号等级分布**: 在一个考核周期（每周）内，团队中处于不同`账号等级`（S, A, B, C）的成员数量分布。数据来源于`ADHD账号评级规则`及项目周报/数据中台。
*   **高质量培训成果**: 在一个考核周期（月度）内，该策划成功将一名新助理从零培养至首次达到S级账号标准的次数。

---

### 二、 薪酬元规则 (Compensation Meta-rules)

#### **`XC-7-1`：策划团队业绩提成规则**
1.  **规则**：根据策划所带领的团队在一个月内完成的`整体团订单数量`，计算其团队业绩提成。
2.  **概念定义**：
    *   `团队订单`：指归属于该策划团队下所有助理的、已确认成交的订单。
3.  **参数**：
    *   `参数1`：每笔团队订单提成（元/单）。
4.  **计算方法**：
    *   `薪酬 = 参数1 * 整体团订单数量`
5.  **数据来源**：`订单信息表`
6.  **相关注释**：此规则激励策划对整个团队的最终业务产出负责。

#### **`XC-7-2`：策划团队等级维护激励规则**
1.  **规则**：根据策划所带领团队的成员每周的`账号等级`分布，发放对应的周度管理激励。
2.  **概念定义**：
    *   `团队成员评级`：指在每周固定时间点，对团队内每一位助理的账号进行的等级评定。
3.  **参数**：
    *   `参数1`：S级成员周度管理激励（元/人/周）。
    *   `参数2`：A级成员周度管理激励（元/人/周）。
    *   `参数3`：B级成员周度管理激励（元/人/周）。
4.  **计算方法**：
    *   `周度激励 = (参数1 * S级成员数) + (参数2 * A级成员数) + (参数3 * B级成员数)`
    *   `月度激励 = Σ (每周的周度激励)`
5.  **数据来源**：`ADHD账号评级规则`、项目周报/数据中台
6.  **相关注释**：此规则激励策划关注团队成员的日常表现和内容质量，提升团队整体实力。

#### **`XC-7-3`：策划培训激励规则**
1.  **规则**：根据策划完成的`高质量培训成果`数量，发放一次性激励奖金。
2.  **概念定义**：
    *   `新S级助理培养`：指一名新助理在该策划的指导下，首次达成S级账号标准。
3.  **参数**：
    *   `参数1`：新S级助理培养奖金（元/人）。
4.  **计算方法**：
    *   `薪酬 = 参数1 * 高质量培训成果数量`
5.  **数据来源**：人力资源系统-培训与绩效模块
6.  **相关注释**：此规则直接激励策划在人才培养和梯队建设上的贡献。

### 三、 薪酬卡片 (Salary Cards)

#### **ADHD策划（培训节点）薪酬卡片**
*   **项目**：ADHD督导业务
*   **角色**：策划（培训节点）

**1. 薪酬构成**：
*   **团队业绩提成**: 引用元规则 `XC-7-1`；`参数1` = `5` 元/单。
*   **团队等级维护激励**: 引用元规则 `XC-7-2`，参数设定如下：
    *   `参数1` (S级成员) = `25` 元/人/周
    *   `参数2` (A级成员) = `15` 元/人/周
    *   `参数3` (B级成员) = `5` 元/人/周
*   **培训激励**: 引用元规则 `XC-7-3`；`参数1` = `300` 元/人。

**2. 考核规则**：
*   **团队S级成员占比**: 月末团队S级成员数需占团队总人数的50%以上。
*   **团队健康度**: 团队内不得出现成员连续两周为C级账号的情况。
*   **培训成功率**: 年度新入职助理的S级转化率需大于60%。

### 四、 核心指标与薪酬测算

本章节旨在基于已定义的薪酬规则，对ADHD策划岗位的月度薪酬进行合理性测算。

#### **1. 单节点月度到手薪酬测算**

*   **ADHD策划到手薪酬** = `团队业绩提成` + `团队等级维护激励` + `培训激励`
    
    *   **示例**: 一位策划带领一个5人助理团队，在一个月（假设为4周）内，其团队业绩如下：
        *   **团队总订单**: 完成 `80` 笔。
        *   **团队等级分布**:
            *   第一周: 2个S级, 2个A级, 1个B级
            *   第二周: 3个S级, 1个A级, 1个B级
            *   第三周: 3个S级, 2个A级, 0个B级
            *   第四周: 4个S级, 1个A级, 0个B级
        *   **培训成果**: 本月成功将 `1` 名新助理培养至S级。
    *   **薪酬计算**:
        *   **团队业绩提成**: 80单 × 5元/单 = **400元**
        *   **团队等级维护激励**: 
            *   周一: (2×25)+(2×15)+(1×5) = 85元
            *   周二: (3×25)+(1×15)+(1×5) = 95元
            *   周三: (3×25)+(2×15) = 105元
            *   周四: (4×25)+(1×15) = 115元
            *   激励合计: 85+95+105+115 = **400元**
        *   **培训激励**: 1人 × 300元/人 = **300元**
        *   **合计月度薪酬**: 400 + 400 + 300 = **1100元**
    *   **说明**：此薪酬为基础薪酬，体现了策划在团队业绩、日常管理和人才培养三方面的综合贡献。
