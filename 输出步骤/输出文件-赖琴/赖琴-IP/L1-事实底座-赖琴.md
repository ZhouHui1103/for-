## 上送给 L2 的交接键

- **分群草图**:
    - 无经验的年轻创业者
    - 拿到短期结果但迷茫的创业者
    - 遇到增长瓶颈的成熟创业者
- **真实触点列表**:
    - 新媒体内容 -> 私域 -> 1v1咨询
- **SKU事实**:
    - 高客单价投融资咨询服务
- **决策Top**:
    - 解决增长焦虑，突破瓶颈
    - 获得更高维度的投资人视角
    - 获得专业的方向引领和“托举”
- **异议Top**:
    - 怀疑顾问的真实能力和价值
    - 过于自信，认为自己不需要指导
- **场景&情绪句**:
    - (创业者) "我总觉得哪里不对劲，我不想被超越" -> 焦虑、不安全感
    - (创业者) "他对自己是没有认知的，他觉得我当前要赚快钱" -> 短视、认知局限
