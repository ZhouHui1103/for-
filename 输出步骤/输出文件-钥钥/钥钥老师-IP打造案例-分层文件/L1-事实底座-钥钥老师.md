## 2. 产品与成交路径表

### **核心产品信息 (加盟模式)**
- **产品/服务名称**: 语音直播公会加盟 / IP操盘手加盟
- **产品定义**: 模式加盟，提供从0到1的业务搭建、培训、管理和客户支持体系。
- **核心卖点/被认可点**:
    - "我会把我所有...（经验）...我给他们造梦了" (录音2, 00:17:56)
    - "三个月让你保证一百万流失（水），你把这个钱赚回来" (录音2, 00:19:24)
    - "我卖所有东西，我都在卖模式" (录音1, 01:10:10)
    - 教会操盘手做经营，获得可持续的“税后收入”。 (录音1, 00:47:40)
- **被吐槽点**: 语音直播业务合规风险大，行业萎缩，情绪消耗大。 (录音2, 00:32:03)

### **产品矩阵**
1. **低价引流品**:
    - **名称**: 线下沙龙
    - **价格**: 299元/人
    - **目的**: 筛选意向用户，为高价产品铺垫。 "结婚就是百分之百成交的" (录音2, 00:56:22)
2. **知识付费产品**:
    - **名称**: 主播21天成长营 / 操盘手课程
    - **价格**: 4980元 (主播)
    - **目的**: 技能培训，筛选意向加盟者。 (录音2, 00:33:31)
3. **中端陪跑产品**:
    - **名称**: 公会梳理运营 / 半年陪跑
    - **价格**: 9800元 / 19800元
    - **目的**: 深度服务，建立更高信任。 (录音2, 00:33:31)
4. **核心加盟产品**:
    - **名称**: 公会加盟合作
    - **价格**: 9.8万元
    - **目的**: 深度绑定，模式复制，长期流水服务费分成。 (录音2, 00:19:24)

### **用户来源**
- **渠道名称**: 内部转化 (自有主播/管理)
- **占比**: 早期核心来源
- **获客成本**: 低 (高信任基础)

- **渠道名称**: 同行转介绍
- **占比**: 持续来源
- **获客成本**: 中

- **渠道名称**: 公域流量 (抖音/视频号等)
- **占比**: 增长潜力
- **获客成本**: 高

### **成交方式/链路**
- **方式名称**: 内部转化链路
- **操作流程**: 1v1沟通 -> 描绘蓝图 (成为老板) -> 承诺结果 (3个月回本) -> 签订9.8万加盟协议。

- **方式名称**: 公域转私域成交链路
- **操作流程**: 短视频内容 -> 添加微信 -> 参与299元沙龙 -> 筛选高意向用户 -> 推荐更高阶产品/加盟。
