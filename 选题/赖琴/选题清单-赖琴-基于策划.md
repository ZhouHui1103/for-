# 选题清单-赖琴-基于策划

## 选题卡片 1/5：立人设，定调性

- **话题词**: `#创业`, `#认知`, `#投资人视角`
- **选题**: 为什么说，现在的创业者比10年前更需要“托举”？
- **一句话Brief**: 提出“当代创业者更脆弱”的核心论点，指出由于创业环境更严峻、创业者认知精力不足等原因，他们比过去任何时候都更需要投资人或专业顾问的“托举”。
- **开头前5s抓人点 (Hook)**: "10年前创业，靠的是胆子大；现在创业，靠的是有人托着你走。"
- **人设落脚点**: 确立IP作为“投融资顾问”和“行业引领者”的专业、权威形象。
- **结尾钩子 (Call to Action)**: "你认为，这个时代的创业，更难了还是更容易了？"
- **目标受众编号**: B级：泛商业人群 / 早期创业者
- **属性**:
  - **维度 (可多选)**: [x] 流量 [x] 人设
  - **流量类型 (单选)**: [x] 泛流量 [ ] 垂直流量
- **[策略对齐]**: 此选题用于**奠定IP的核心角色——“托举者”**，一上来就输出核心价值观，建立与市场上其他商业博主的差异化定位。

## 选题卡片 2/5：打痛点，引共鸣

- **话题词**: `#创业`, `#焦虑`, `#解决方案`
- **选题**: 当你觉得“不对劲”的时候，你就要去看“医生”了。
- **一句话Brief**: 将创业中遇到的增长瓶颈、方向迷茫等问题，比作“生病”，鼓励创业者在感知到问题时，要像看医生一样，主动寻求外部专业顾问的“诊断”和帮助。
- **开头前5s抓人点 (Hook)**: "我总觉得哪里不对劲，总觉得别人很快就要超越我了...如果你也有这种感觉，别硬扛，你该去看‘医生’了。"
- **人设落脚点**: 塑造专业、可信赖、能解决问题的“创业顾问”形象。
- **结尾钩子 (Call to Action)**: "你最近一次感觉“不对劲”，是什么时候？"
- **目标受众编号**: A级：焦虑的早期创业者
- **属性**:
  - **维度 (可多选)**: [x] 流量 [x] 人设 [x] 转化
  - **流量类型 (单选)**: [x] 泛流量 [ ] 垂直流量
- **[策略对齐]**: 此选题**精准命中核心目标用户（焦虑的早期创业者）的痛点**，用“看医生”的比喻降低沟通成本，直接引导转化。

## 选题卡片 3/5：给方法，秀肌肉

- **话题词**: `#创业`, `#事业`, `#认知误区`
- **选题**: 别把做业务当成做事业，90%的创业者都死在这上面。
- **一句话Brief**: 深刻剖析“业务”与“事业”的本质区别，警示只顾埋头做业务、赚快钱的创业者，会很快碰到天花板，必须拥有做事业的长期蓝图。
- **开头前5s抓人点 (Hook)**: "你是在做业务还是做事业？一句话帮你分清：业务是让你生存的动作，而事业是让你持续存在的蓝图。"
- **人设落脚点**: 塑造具有顶层设计能力的“战略家”形象，提供超越具体执行的认知价值。
- **结尾钩子 (Call to Action)**: "你现在做的事，是业务，还是事业？"
- **目标受众编号**: A级：焦虑的早期创业者 / 成熟创业者
- **属性**:
  - **维度 (可多选)**: [ ] 流量 [x] 人设 [x] 转化
  - **流量类型 (单选)**: [ ] 泛流量 [x] 垂直流量
- **[策略对齐]**: 输出IP的核心方法论之一（事业VS业务），**展现IP在“顶层设计”层面的专业能力**，筛选出认可此价值观的高质量用户。

## 选题卡片 4/5：亮身份，提逼格

- **话题词**: `#投资`, `#识人`, `#创始人`
- **选题**: 我们投的不是业务，是你身上的“冠军特质”。
- **一句话Brief**: 揭示投资人的真实视角，即相比于当下的业务数据，他们更看重创始人身上是否具备能够穿越周期的“特质”，并以此来判断项目的最终天花板。
- **开头前5s抓人点 (Hook)**: "你以为投资人看的是你的BP和流水吗？错了，我们第一眼看的，是你身上有没有这几种味道。"
- **人设落脚点**: 强化“投资人”的专业身份，提供识人识己的高维认知。
- **结尾钩子 (Call to Action)**: "你认为，一个成功的创始人，最重要的特质是什么？"
- **目标受众编号**: A级：焦虑的早期创业者 / S级：寻求融资者
- **属性**:
  - **维度 (可多选)**: [ ] 流量 [x] 人设 [x] 转化
  - **流量类型 (单选)**: [ ] 泛流量 [x] 垂直流量
- **[策略对齐]**: 强化IP的“投资人”身份标签，**输出独特的、高维度的识人方法论**，进一步提升IP的专业势能和稀缺性。

## 选题卡片 5/5：树标杆，做案例

- **话题词**: `#创业`, `#转型`, `#战略`
- **选题**: 从服务C端到服务B端，这家公司只用8个月，流水翻了10倍。
- **一句话Brief**: 通过一个从C端转型B端并获得爆发式增长的真实案例，为陷入C端增长瓶颈的创业者，提供一个可借鉴的、极具诱惑力的转型路径。
- **开头前5s抓人点 (Hook)**: "当所有人都还在C端流量里厮杀的时候，这个95后女孩，靠一次转型，只用了8个月，月流水就从十几万做到了几百万。"
- **人设落脚点**: 展现IP指导企业完成战略转型的实战能力，作为最硬核的信任状。
- **结尾钩子 (Call to Action)**: "你的业务，适合从C端转向B端吗？私信我，聊聊你的困惑。"
- **目标受众编号**: A级：焦虑的早期创业者 / 遇到瓶颈的成熟创业者
- **属性**:
  - **维度 (可多选)**: [ ] 流量 [x] 人设 [x] 转化
  - **流量类型 (单选)**: [ ] 泛流量 [x] 垂直流量
- **[策略对齐]**: 这是战略规划中明确要求的**“标杆案例”内容，是IP实战能力最直接的证明**，作为最强的信任状和转化工具。
