# 爆款文案结构公式

## 00 通用类
**结构：** 钩子开头 + 塑造期待 + 解决方案 + 结尾

**原理：**
口播类视频的通用型结构，可以根据具体情况对4个环节进行增删；比如在「塑造期待」后再次加入「钩子」强化期待的效果，也一些情况会直接省略结尾文案等等！通用类公式的变化会在具体的公式案例中进行解析。

---

## 01 结构一：现象危害模型
**结构：** 现象 + 危害 + 原因 + 解决办法
(目标人群+痛点/共鸣式钩子)

**案例解析：**
1.  **现象(圈定/目标人群+钩子):** 黄皮女孩买衣服太难选颜色了
    **危害:** 挑错了颜色显的脸又黑又老
    **原因:** 但皮肤颜色很多时候又是天生的
    **解决办法:** 这三种颜色不仅显白还超显气质

2.  **现象:** 很多女孩其实不胖/但是就是小腿粗
    **危害:** 穿裤子特别不好看
    **原因:** 其实跟你长期久坐有关系
    **解决办法:** 每天做这三个动作一周就能瘦一圈

## 02 结构二：信任背书模型
**结构：** 炸裂式的开头 + 人设信息 + 高密度的信息盘点 + 互动式结尾
(反差类钩子, 信任背书/塑造期待, 解决方案)

**案例解析：**
1.  **炸裂式的开头/吸引注意力:** 不到100元就能搞定的一些智能家居
    **信任背书:** 同事们体验后都在要链接
    **罗列相关信息:** □□品牌智能读书灯□□元
    **互动结尾:** 宝子们还有哪些宝贝评论区见!

2.  **炸裂式的开头/吸引注意力:** 还有人不知道手机拍照是可以赚钱的吗
    **信任背书:** 好多家人已经来报喜了
    **解决方案:** 将拍到的照片上传□□网站
    **互动结尾:** 关注我给生活增添更多惊喜

## 03 结构三：利益前置模型
**结构：** 积极结果 + 获得感 + 方案 + 互动式结尾
(利益性结果前置)

**案例解析：**
1.  **积极目标前置:** 家人们!我爆单啦!
    **获得感:** 上两期讲的爆款公式火了
    **方案:** □□用完之后/粉丝直接从□□变成□□,当月单个广告变现□□一条带货视频就分佣□□

2.  **获得感:** 你只要按照我以下的脚本公式去写/我相信10条之内必有一个小的爆款

3.  **价值/方案:** 为了方便记忆我总结了以下四条 (步骤1/步骤2/步骤3/步骤4) 带货视频可以展示陈述卖点1/卖点2/卖点3

4.  **口号式结尾:** 关注我带你低成本变美

## 04 结构四：金句佐证模型
**结构：** 列金句 + 佐证 + 列金句 + 佐证
(连续递进式)

**案例解析：**
1.  **列金句:** “方言才是创作的母体啊!”
    **佐证:** “芙蓉路的妹陀,岳阳的伢子” 你们听,这两句方言一出,就勾勒出两地青年的画像, 长沙女孩子个子小小,绵软可爱,岳阳的小青年混不吝,充满了干劲,方言带来的画面感极易引起共鸣!

2.  **列金句:** “未来十年,下沉市场,将诞生一批杰出的文艺作品!”
    **佐证:** “毕赣搞出了路边野餐,五条人搞出了阿珍爱上了阿强” “刘爽搞出了漠河舞厅”为啥呢?一线城市已经搞不出作品来了!他们的月收入只要超过了3万,灵魂就洋化了,双脚不沾地!

## 05 结构五：行业揭秘模型
**结构：** 行业揭秘 + 塑造期待 + 解决方案

**案例解析：**
1.  **揭秘型钩子:** 讲一个专业编导一般都不太愿意讲的爆款内容脚本结构
2.  **期待塑造:** 我不敢保证说你使用完之后能百分之百出爆款,但是你的完播率翻上一倍,这个信心我还是有的因为基本上我的爆款视频都是按照这个结构构写的,其他你们知道的头部知识博主的脚本基本都在套这个结构,你大可以去拆解其中的文案!
3.  **解决方案:** 这个结构的名字叫“PREP结构”我是跟一个演讲高手学的!

## 06 结构六：利益传递模型
**结构：** 利益传递 + 强化期待 + 解决办法 + 结尾

**案例解析：**
1.  **利益传递类钩子:** 教你一条万能的短视频文案公式
2.  **强化期待:** 你刷到的很多点赞几百W的视频文案都是这样写的,其实一点也不难,直接套用就好而且非常百搭不管你是做美食的,美妆的,穿搭的,还是讲育儿的,讲情感的都可以用这条公式,记得点赞收藏!

## 07 感性观点分享格式
**结构：** 事实 + 个人感受 + 发现问题 + 引出观点 + 讲故事 + 总结观点

**案例解析：**
*   **事实:** 最近发生了一个什么事
*   **个人感受:** 让我产生了什么感觉
*   **发现问题:** 这背后存在一个什么问题
*   **引出观点:** 我觉得问题是什么?
*   **讲故事:** 一个能证明我观点的故事
*   **总结观点:** 所以我们应该怎样怎样

---

## 08 经典的知识分享4段结构
**结构：** 问题描述 + 问题的拆解 + 答案描述 + 答案拆解

**案例解析：**
1.  **问题描述:** 我知道你遇到了什么问题
2.  **问题的拆解:** 我已经找到了你的问题所在
3.  **答案描述:** 我来给你看看我的解决方案
4.  **答案拆解:** 为什么这个办法能解决你的问题
