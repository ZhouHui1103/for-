# 搭建流程SOP - 简要

### 模块部分
1. 梳理模块清单（不同层次） - 模块清单表
2. 梳理模块业务故事 + 业务对象概览（不同层次）
3. 梳理模块计费标准 + 考核评级标准 + 财务模型估算
4. 部分人力节点文档写作梳理
    

### 本体论部分
1. 梳理整体业务元数据：对象明细 + 场域明细 + 相关模板
2. 推导对应权限逻辑 + 表格结构设计（ER）+ 相关报表设计
3. 搭建部署完整表格系统
    

### 流程梳理
1. 生成对应的数据部分L3文档
2. 陪跑梳理对应业务流程细节
3. 梳理对应的自动化流程清单
4. 部署对应自动化流程（数据+业务操作）
    

#### 执行部署
1. 通过人力流程操作
2. 部署跟踪
3. 财务核算验证校准
