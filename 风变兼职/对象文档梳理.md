1. 任务种类：表达特定任务类型，可以是二级表达
	1. 主键：编号-编号（一级任务-二级任务）
	2. 属性：被动状态
	3. 状态：待打样、已打样
	4. 分类：一级种类（AI做图、笔记发送、素材制作）
	5. 其他关联：无
	6. 其他字段：关联SOP文档链接、定价价格
	7. 人员：打样负责人（培训节点）
	8. 时间：提出时间、打样完成时间
2. 单个任务：表达某个具体的特定任务（这里待定，待看原始文档）
	1. 主键：任务编号（日期 + 序号）
	2. 属性：主动状态
	3. 状态：待排期、进程中、已完成、已撤销
	4. 分类：任务种类（关联）
	5. 其他关联：无
	6. 其他字段：
	7. 人员：HRBP、内审、外审、需求提出
	8. 时间：提出时间、首次排期时间、结束时间（完成/撤销）
3. 任务进程：某个具体的次数
	1. 主键：进程编号：任务编号 + 次数（1/2/3）
	2. 属性：主动状态
	3. 状态：进行中、内审中、外审中、返工中、已弃单、已验收
	4. 种类：任务种类（计算）
	5. 其他关联：单个任务
	6. 其他字段：
	7. 人员：HRBP、执行节点、培训节点（计算）、内审、外审、需求提出
	8. 时间：排期时间、结束时间
4. Task对象（也可以暂时考虑放弃 - 如果是人工操作的话 - 自动化则需要）：
	1. 主键：Task编号：任务进程编号 + 序号（1/2/3）
	2. 属性：主动状态
	3. 状态：已分配、已完成、已放弃
	4. 种类：内审、外审、初次执行、返工修改
	5. 其他关联：任务进程
	6. 其他字段：
	7. 人员：HRBP、负责节点
	8. 时间：排期时间、结束时间
5. 节点对象：
	1. 主键：人员编号
	2. 属性：被动状态
	3. 状态：生效中、已失效
	4. 种类：执行节点、培训节点、内审节点、HRBP、外审节点
	5. 其他关联：Leader节点（如培训节点）
	6. 人员：某个个体
	7. 时间：生效时间、失效时间
